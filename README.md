## Projekt aufsetzen
Python Version: 3.6

### Anaconda

```
conda create --name ml_workshop python=3.6
conda activate ml_workshop
```

Jupyter Notebook aufsetzen
```
conda install -c anaconda ipykernel
python -m ipykernel install --user --name=workshop
```
